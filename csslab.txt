clamp(MIN, PREFERRED, MAX)
Think of it like:

“Value ko MIN se chhota mat jaane do,
MAX se bada mat jaane do,
beech mein screen ke hisaab se smoothly badhne do.”

❌ Old way (media queries):

p { font-size: 16px; }

@media (min-width: 768px) {
  p { font-size: 20px; }
}

@media (min-width: 1200px) {
  p { font-size: 24px; }
}


✅ New way (clean + modern):

p {
  font-size: clamp(16px, 4vw, 24px);
}
